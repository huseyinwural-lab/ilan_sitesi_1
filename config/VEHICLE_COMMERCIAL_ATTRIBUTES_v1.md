# Vehicle Commercial Attributes v1

**Scope:** Vans, Trucks (Ticari Araç).

## 1. Specs
-   **Key:** `comm_vehicle_type`
    -   **Type:** Select
    -   **Options:** Van (Panelvan), Truck (Kamyon), Pickup (Kamyonet), Bus (Otobüs).
-   **Key:** `load_capacity_kg`
    -   **Type:** Number
    -   **Unit:** kg
    -   **Filter:** Range.
-   **Key:** `box_type`
    -   **Type:** Select
    -   **Options:** Closed Box, Open Bed, Refrigerated (Frigo), Tipper (Damper).
