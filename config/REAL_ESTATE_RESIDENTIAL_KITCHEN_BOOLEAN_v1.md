# Real Estate Residential Kitchen Boolean v1

**Goal:** Simplify Kitchen attribute.

## 1. Attribute Definition
-   **Key:** `has_kitchen` (Replaces complex kitchen enums if any).
-   **Type:** Boolean (Checkbox).
-   **Filter:** Checkbox ("Kitchen Available" / "Küche vorhanden").

## 2. Labels
-   **EN:** Kitchen
-   **TR:** Mutfak
-   **DE:** Einbauküche (EBK) / Küche
-   **FR:** Cuisine

## 3. Logic
-   **Default:** Checked (True) for most Residential Listings.
-   **Optional:** Might be False for raw/unfurnished apartments or specific studio types.
