# Admin UI Data Acceptance Checklist

**Goal:** Ensure Admin screens look "alive".

## 1. Member Lists
- [ ] **Individual:** Verify "Verified" badges appear on 70% of rows.
- [ ] **Individual:** Verify Country flags (DE/TR/FR) are visible.
- [ ] **Dealer:** Verify "Tier" column shows Standard/Premium/Enterprise labels.
- [ ] **Dealer:** Verify "Active Package" column is populated.

## 2. Dashboard
- [ ] **Charts:** "Listings by Status" pie chart is not empty.
- [ ] **Counts:** "Total Users" > 20.

## 3. Moderation
-   [ ] **Queue:** "Pending" tab has items to review.
-   [ ] **History:** "Rejected" tab has items with rejection reasons.
