## Reports Audit Integration

### Event Types
- `REPORT_CREATED`
- `REPORT_STATUS_CHANGE`

### Alanlar (önerilen)
- `report_id`, `listing_id`, `country_code`
- `previous_status`, `new_status` (status change için)
- `note` (status change)
- `reason` (created)
- `admin_user_id` / `user_id` / `user_email`
