## Admin Reports UI Wiring

### Route
`/admin/reports`

### Özellikler
- Filtreler: status, reason, listing_id
- Pagination
- Satır aksiyonları: Detay, Moderation’a Git, Listing’e Git
- Detay modalında status değişimi + not

### Notlar
- Country-scope URL paramı (`?country=XX`) API çağrılarına taşınır
