# FAZ-L1: Seed Listing Operation Plan

## 1. Goal
Reach **1,000 Active Listings** within 14 days to solve the "Chicken and Egg" problem.

## 2. Target Markets (DE Focus)
*   **Berlin**: 400 Listings (Focus: Apartments for Rent)
*   **Munich**: 300 Listings (Focus: Premium Real Estate)
*   **Hamburg**: 200 Listings
*   **Cologne**: 100 Listings

## 3. Seed Sources
1.  **Manual Entry (Internal Team)**:
    *   Target: 50 listings/day per person.
    *   Source: Public domain data, partner networks.
2.  **Dealer Partnerships (Anchor Tenants)**:
    *   Target: 5 key dealers in Berlin.
    *   Incentive: 6 months free "Enterprise" plan + Showcase.
3.  **Data Migration (Import)**:
    *   Tool: CSV Bulk Uploader (Admin only feature).

## 4. Quality Control
*   Every seed listing MUST have at least 3 photos.
*   Description must be > 50 words.
*   Price must be realistic (within 20% of market avg).
