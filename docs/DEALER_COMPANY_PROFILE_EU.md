# DEALER_COMPANY_PROFILE_EU

## Amaç
EU ticari gerekliliklere uygun şirket profili.

## Alanlar
- Firma adı (zorunlu)
- VAT ID (VIES format kontrol)
- Ticaret sicil no
- Yetkili kişi
- Şirket adresi (EU format)
- Logo upload
- AB tüketici bilgilendirme metni

## Bileşenler
- Firma formu
- VAT doğrulama feedback
- Logo upload

## Empty state
- Zorunlu alanlar placeholder

## Success/Error
- VAT format invalid → error
