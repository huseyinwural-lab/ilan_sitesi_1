# User Panel V1 Scope Lock

## 1. Vision
To provide a frictionless, "3-click-to-publish" experience for listing creation and a unified dashboard for managing marketplace activities.

## 2. In-Scope (V1)

### 2.1. Core User Flows
*   **Onboarding**: Registration (Email/Phone), Type Selection (Individual/Business).
*   **Listing Management**: Create (Wizard), Edit, Delete, Repost (Renew).
*   **Messaging**: Inbox list, Chat detail, Push notifications.
*   **Account Settings**: Profile photo, Password change, Notification prefs.

### 2.2. User Types
*   **Individual**: Free tier logic, Basic profile.
*   **Business (Dealer)**: Listing limits, Company profile, Bulk actions (Future-ready UI).

### 2.3. Monetization Integration
*   **Premium Purchase**: "Buy Boost" flow for existing listings.
*   **Subscription**: View current plan status (Business).

## 3. Out-of-Scope (Deferred to V1.1)
*   **Bulk Upload**: CSV/XML import for dealers.
*   **Sub-accounts**: Multi-user access for one business account.
*   **Advanced Analytics**: Detailed graphs for listings (Basic view count is IN).
*   **Affiliate Dashboard**: Moved to separate Affiliate Portal (P18 scope).

## 4. Quality Standards
*   **Mobile First**: All screens designed for 375px width first.
*   **Performance**: Time to First Interaction < 1.5s on 4G.
