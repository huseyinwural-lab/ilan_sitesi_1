# Dealer Public Profile UI Spec

## 1. Header (Storefront)
*   **Banner**: Cover image (Optional).
*   **Logo**: Square profile picture.
*   **Badges**: Verified Checkmark, "Premium Dealer".
*   **Actions**: "Call", "Message", "Website".

## 2. Info Tab
*   **About**: Description text.
*   **Map**: Static map image of address.
*   **Contact**: Phone, Email (masked), Address.
*   **Hours**: Opening hours (if available).

## 3. Listings Tab (Inventory)
*   **Search In Store**: Input field to filter dealer's items.
*   **Grid**: Standard Search Card components.
*   **Filters**: Simplified version of main search (Category, Price).

## 4. Footer
*   "Report this dealer".
*   "Share this profile".
