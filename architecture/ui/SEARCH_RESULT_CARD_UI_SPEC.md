# Search Result Card Spec

## 1. Elements
*   **Image**: Cover photo. Aspect ratio 4:3.
*   **Badges (Overlay)**:
    *   `Premium`: Gold badge top-left.
    *   `Commercial`: "Ticari" badge bottom-right.
*   **Content**:
    *   **Price**: Large font, primary color.
    *   **Title**: 2 lines max, ellipsis.
    *   **Meta**: `City / District` • `Room` • `m²`.
*   **Footer**:
    *   **Date**: "2 days ago".
    *   **Actions**: Heart icon (Favorite).
