# User Core Module

Bu modül, son kullanıcının kimlik, profil ve güvenlik ayarlarını yönetir.

## Sorumluluklar
*   Kullanıcı profili (Self-Service)
*   KYC Durum Takibi
*   İletişim Tercihleri

## Dizin Yapısı
*   `routers/user_panel/me.py`: Profil endpointleri.
*   `schemas/user_core.py`: Pydantic modelleri.
