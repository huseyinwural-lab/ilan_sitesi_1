# ADMIN_NAVIGATION_CONSISTENCY_RULES

## Zorunlu Kurallar
1) **Breadcrumb zorunlu**
- Admin içinde her sayfa üstte breadcrumb gösterir.

2) **Active state net**
- Sidebar aktif link state’i belirgin.

3) **Section header görsel ayrımı**
- Her page: Başlık + kısa açıklama.

4) **Sidebar collapse (desktop)**
- Sidebar daraltılabilir.
- Collapse durumda icon-only.

## UX Detayları
- Menüler domain grupları olarak ayrılmalı.
- “Coming Soon” sayfalar linklenebilir ama disabled/etiketli olmalı.

## Teknik Kurallar
- Route’lar /admin/* altında.
- RBAC menü filtreleme: Layout seviyesinde.
