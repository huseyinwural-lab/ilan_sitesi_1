# Customer Support Workflow v1

## 1. Ticket Categories
*   **Billing**: "Invoice not received", "Double charge".
*   **Listing**: "Why was my listing rejected?", "Cannot upload photos".
*   **Account**: "Forgot password", "Dealer verification pending".
*   **Safety**: "Report scammer", "Fake listing".

## 2. Triage Process (Level 1)
*   **SLA**: 24 hours.
*   **Tools**:
    *   `AuditLog` dashboard.
    *   `Impersonation` (Admin feature to view as user - PENDING).
*   **Response**: Templates (Macros) for common issues.

## 3. Escalation (Level 2)
*   **Technical**: Bug reports -> GitHub Issue.
*   **Compliance**: Fraud reports -> Trust & Safety Officer (P27 Risk Engine review).
