# Dealer Pricing Matrix v2 (Final)

## 1. Package Structure (Monthly Subscription)

| Feature | Basic | Pro | Enterprise |
| :--- | :--- | :--- | :--- |
| **Listing Quota** | 10 Listings | 50 Listings | Custom (100+) |
| **Showcase Limit** | 0 | 5 Listings | Custom |
| **Storefront** | Basic | Branded (Logo + Cover) | Premium Branding |
| **Analytics** | Basic Stats | Full Lead Access | API Access |
| **Support** | Email | Priority Email | Dedicated Account Manager |
| **Price (Monthly)** | €49.00 | €149.00 | Contact Sales |

## 2. Add-ons (Over-Quota)
*   **Extra Listing**: €5.00 / listing / month.
*   **Extra Showcase**: €15.00 / listing / month.

## 3. Billing Rules
*   **Cycle**: Monthly, auto-renew.
*   **Upgrade**: Immediate (Prorated charge).
*   **Downgrade**: Effective at next cycle start.
*   **Grace Period**: 3 days for failed payment before listings suspend.
