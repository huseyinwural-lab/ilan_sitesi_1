# Search Result Card Specification

## 1. Visual Components (Frontend Contract)

### 1.1. Media
*   **Thumbnail**: Aspect Ratio 4:3.
*   **Count**: "1/15" overlay.

### 1.2. Header
*   **Price**: Bold, Large. Format: `€ 450.000`.
*   **Title**: Truncated to 2 lines.

### 1.3. Attributes (Specs)
*   **Line 1**: `85 m²` • `2+1` • `3. Kat`.
*   **Line 2**: `Berlin / Mitte`.

### 1.4. Badges (Compliance)
*   `Ticari` / `Commercial`: Gray badge.
*   `Bireysel` / `Private`: No badge or distinct color.
*   `Premium`: Gold border or star.

### 1.5. Footer
*   **Time**: "2 gün önce".
*   **Action**: Favori ikonu.
