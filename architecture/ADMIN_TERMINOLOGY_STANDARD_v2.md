# Admin Terminology Standard v2

**Language:** Turkish (TR)
**Scope:** Admin Panel UI Labels

| Technical Term | **Admin UI Term (TR)** | Description |
| :--- | :--- | :--- |
| **Dealer** | **Ticari Üye** | Business entity (Galeri, Emlakçı). |
| **Individual** | **Bireysel Kullanıcı** | Private person listing items. |
| **User** | **Kullanıcı** | System admin or staff member. |
| **Feature Flag** | **Sistem Özelliği** | Toggleable system capability. |
| **Attribute** | **İlan Alanı** | Dynamic fields (e.g., KM, Room Count). |
| **Premium Product**| **Paket** | Subscription or visibility booster. |
| **Tier** | **Üyelik Seviyesi** | Rate limit capacity (Standard/Premium). |
| **Audit Log** | **Sistem Logu** | Action history. |
| **Rate Limit** | **İşlem Limiti** | API throttle threshold. |
